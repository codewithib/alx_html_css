ALX Interface Desktop Version

I created a desktop version of the savanna ALX Interface to practice and enhance my skills with CSS Flexbox.

This is a single-page project, and while I haven’t delved deeply into advanced CSS concepts yet, my main focus was to understand and apply Flexbox effectively. This project was a great way to test my understanding and experiment with layout techniques.

Key Takeaways
	•	Gained practical experience with Flexbox for building responsive layouts.
	•	Improved my overall understanding of CSS styling.
	•	Learned how to structure and style components in a cohesive manner.
